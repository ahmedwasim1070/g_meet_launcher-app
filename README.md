# Google Meet Launcher

**Package Name:** `com.ahmedwasim1070.gmeet-manager-app`

Google Meet Launcher is a single-stop desktop solution engineered specifically for entrepreneurs to manage schedules, watch meeting links, and launch Google Meet sessions instantly without browser clutter. Built with modern web technologies and packaged via Tauri, it provides a native, seamless experience with an incredibly small memory footprint.

## ✨ Key Features

- **One-Click Launching:** Instantly start or join your scheduled meetings.
- **Smart Scheduling:** Built-in scheduler to keep your itinerary organized directly from your desktop.
- **Seamless Background Management:** Configure and apply your meeting backgrounds easily without digging through browser settings.
- **Lightweight & Fast:** Powered by Tauri and Rust, meaning it consumes a fraction of the RAM compared to standard Electron apps.
- **Native Desktop Experience:** Runs smoothly on Windows with system-level notifications and clipboard watching.

## 🛠 Tech Stack

- **Core Framework:** Tauri (Rust)
- **UI Framework:** React (with Vite)
- **Language:** TypeScript & Rust
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
